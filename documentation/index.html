jQuery Calx Overview

jQuery Calx is an excel calculation engine that wrapped as jQuery plugin, It is useful for building calculation in html page using excel-like formula. You can simply define the cell address in data-cell attribute, calculation formula in data-formula attribute, result formatting rule in data-format attribute, and let jQuery Calx do the magic.

Once jQuery Calx is initialized, it will respond to any change that occur in cell's element, and do the calculation based on the cell value and formula, let's see below example: 

    <div id="sheet">
        <input type="text" data-cell="A1"> <br>
        <input type="text" data-cell="A2"> <br>
        <input type="text" data-cell="A3"> <br>
        <input type="text" data-cell="A4" data-formula="SUM(A1:A3)">
    </div>

    <script>
        $('#sheet').calx();
    </script>

In the above sample code snippet, A4 wil display whatever result of SUM(A1:A3), any change occured in A1, A2, or A3 will be represented in A4 since A4 depend on cell ranged from A1 to A3.

If you are familiar with any spreadsheet application like Microsoft Excel or LibreOffice, you will also get familiar with jQuery Calx. We use term sheet for wrapper element like div#sheet where the jQuery Calx is initialized, and cell for all element inside the div#sheet that involved in calculation process, this mean all element that has data-cell and data-formula attribute.
